# Convex Optimization Learning

Interactive notes for building geometric intuition in convex optimization.

## Current chapter

**Convex Functions as Upper Envelopes of Affine Functions**

The page explains:

- the set of affine lower supports of a convex function,
- why the function equals the supremum of those supports,
- the algebraic proof for \(f(x)=x^2\),
- the general differentiable and subgradient cases,
- why the idea matters in economics: prices, duality, shadow values, and global first-order bounds.
