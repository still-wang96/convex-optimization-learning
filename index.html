凸函数为什么可以看作一族仿射函数的上包络？

这一页只讲一个几何直觉：凸函数可以由许多条从下方支撑它的仿射函数“托起来”。 拖动滑块或播放动画，观察有限条支撑线的最大值如何逼近原函数。

动态图像

蓝色曲线是凸函数 \(f(x)=x^2\)。橙色直线是在不同点处的切线， 每一条都是仿射函数，并且都位于曲线下方。绿色折线是这些橙色直线的逐点最大值。

当支撑线越来越密，绿色上包络会越来越贴近蓝色曲线；如果把所有切线都放进去， 它就正好等于原来的凸函数。

要抓住的集合直觉

1. 每条切线是一条“下界”

对凸函数来说，在点 \(a\) 处的切线 \(\ell_a(x)=f(a)+g_a(x-a)\) 不会穿到函数上方。

2. 所有下界组成一个集合

你可以把所有不超过 \(f\) 的仿射函数收集起来： \(\{\ell:\ell(x)\le f(x)\}\)。

3. 对每个 x 取最高的那条

因为这些线都在下面，所以要恢复 \(f(x)\)，就取它们在该点的最大值。

一句话公式

如果 \(f\) 是闭凸函数，那么 

f(x) = sup { a x + b : a t + b <= f(t), for all t }

高维时把 \(a x+b\) 换成 \(a^\top x+b\)。几何上，一维是支撑直线； 高维就是支撑超平面。
