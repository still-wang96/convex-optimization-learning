Convex Functions as Upper Envelopes of Affine Functions

The geometric idea is simple but powerful: a closed convex function can be rebuilt from all affine functions that lie below it. These affine functions are its supporting lines or supporting hyperplanes.

Upper envelopes Supporting hyperplanes

01

The Set Intuition

Fix a convex function \(f\). Now collect every affine function \(\ell(x)=a^\top x+b\) that never goes above \(f\):

\[ \mathcal A_f = \left\{ (a,b): a^\top z+b \leq f(z)\ \text{for every } z \right\}. \]

This is a set of global affine lower bounds. At a particular point \(x\), each pair \((a,b)\in\mathcal A_f\) gives one candidate value \(a^\top x+b\). Since all candidates are below \(f(x)\), the best reconstruction is to take the largest one:

\[ f(x)=\sup_{(a,b)\in\mathcal A_f}\{a^\top x+b\}. \]

So the function is the upper envelope of all its affine lower supports. The word "upper" means the highest value among the supporting affine functions, not a function sitting above \(f\).

03

Algebraic Proof in One Dimension

Take \(f(x)=x^2\). At point \(a\), the tangent line is

\[ \ell_a(x) = f(a)+f'(a)(x-a) = a^2+2a(x-a) = 2ax-a^2. \]

First, this affine function is always below \(x^2\):

\[ x^2-\ell_a(x) = x^2-(2ax-a^2) = (x-a)^2 \geq 0. \]

Therefore \(\ell_a(x)\leq f(x)\) for every \(x\). Second, at \(a=x\), the tangent reaches the curve:

\[ \ell_x(x)=2x^2-x^2=x^2=f(x). \]

Since every tangent lies below \(f\), their supremum cannot exceed \(f\). Since the tangent at \(a=x\) reaches \(f(x)\), the supremum cannot be below \(f(x)\). Hence

\[ x^2=\sup_{a\in\mathbb R}\{2ax-a^2\}. \]

04

General Convex Functions

If \(f\) is differentiable and convex, the first-order convexity inequality says

\[ f(x)\geq f(z)+\nabla f(z)^\top(x-z) \quad \text{for all }x,z. \]

For each fixed \(z\), the right-hand side is affine in \(x\). It is a global lower support. Taking the supremum over all \(z\) gives back \(f\):

\[ f(x) = \sup_z \left\{ f(z)+\nabla f(z)^\top(x-z) \right\}. \]

If \(f\) is not differentiable, replace \(\nabla f(z)\) with any subgradient \(g\in\partial f(z)\):

\[ f(x) = \sup_{z,\ g\in\partial f(z)} \left\{ f(z)+g^\top(x-z) \right\}. \]

05

Why Economists Care

Prices as supporting hyperplanes

In consumer theory and production theory, supporting hyperplanes often behave like prices. A price vector supports an optimal bundle or efficient production plan by separating what is feasible from what is preferred or profitable.

Duality and value functions

Many value functions are convex or concave after the right sign convention. Their affine supports are the objects that appear in dual problems: Lagrange multipliers, shadow prices, and supporting prices.

Local information gives global bounds

Convexity turns a local derivative or subgradient into a global inequality. This is why first-order conditions are so strong in convex optimization and why they are useful in equilibrium and mechanism-design arguments.

What to Remember

A convex function is the highest surface consistent with all of its affine lower supports. Finite supports give a rough polyhedral approximation. All supports give the exact function.
